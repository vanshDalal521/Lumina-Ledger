# 💎 Lumina Ledger

![Lumina Ledger Hero](./hero.png)

## 🌟 Master Your Finances. Illuminate Your Future.

**Lumina Ledger** is a premium, high-aesthetic personal finance tracking system designed to transform how you manage your money. Built with a cutting-edge **Liquid Glass** and **Claymorphism** interface, it offers a seamless, visually stunning experience for tracking daily expenses and staying on top of your monthly budget.

---

## ✨ Why Lumina Ledger?

Tracking your spending shouldn't feel like a chore. Lumina Ledger makes it an experience. Whether you're a student, a professional, or someone who just loves beautiful software, this tool provides the clarity you need with the style you deserve.

### 🚀 Key Features

-   **🎨 Premium Aesthetic**: Experience the future of UI with glassmorphism, smooth gradients, and ambient animated backgrounds that react to your interaction.
-   **📊 Real-Time Analytics**: Instantly see your **Total Budget**, **Total Spent**, and **Remaining Balance** with dynamic updates.
-   **🏷️ Smart Categorization**: Keep your finances organized with intuitive categories like Food, Transport, Books, Fun, and more—each with unique visual badges.
-   **💾 Persistent Memory**: Never lose your data. Lumina Ledger uses advanced `localStorage` to keep your records safe even after you close your browser.
-   **📱 Fully Responsive**: Whether you're on a 4K monitor or a smartphone, the interface adapts perfectly for a "native app" feel.
-   **🗑️ Effortless Management**: Add new expenses in seconds and clear your history with a single tap when you're ready for a fresh start.

---

## 🛠️ Built With

We used a modern, lightweight tech stack to ensure maximum performance and zero bloat:

-   **HTML5**: Pure semantic structure for SEO and accessibility.
-   **CSS3 (Vanilla)**: Advanced styling using CSS Variables, Flexbox, Grid, and the latest glassmorphism techniques.
-   **JavaScript (ES6+)**: High-performance logic for real-time calculations and DOM manipulation.
-   **Icons**: [Phosphor Icons](https://phosphoricons.com/) for that sleek, architectural look.
-   **Typography**: [Google Fonts (Outfit & Inter)](https://fonts.google.com/) for ultimate readability and premium feel.

## 🚀 Getting Started

Getting started with Lumina Ledger is as easy as 1-2-3:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/vanshDalal521/LuminaLedger.git
    ```
2.  **Navigate to the Folder**:
    ```bash
    cd Lumina-Ledger-main
    ```
3.  **Launch the App**:
    Simply double-click `index.html` to open it in your favorite browser. No installation required!
