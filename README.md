💎 Lumina Ledger

**Lumina Ledger** is a premium, high-aesthetic personal finance tracking system designed with a modern **Liquid Glass** and **Claymorphism** interface. It provides a seamless and visually stunning experience for managing your daily expenses and monthly budget.

---

## ✨ Key Features

- **Liquid Glass Design**: A premium UI featuring glassmorphism, smooth gradients, and ambient animated backgrounds.
- **Dynamic Analytics**: Real-time calculation of total budget, total spent, and remaining balance.
- **Smart Categorization**: Categorize your spending (Food, Transport, Books, etc.) with intuitive icons.
- **Persistent Storage**: Uses browser `localStorage` to keep your financial data safe even after a page refresh.
- **Responsive Layout**: Fully optimized for both desktop and mobile devices.
- **Interactive States**: Smooth hover effects and micro-animations using Phosphor Icons.

---

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3 (Vanilla)**: Advanced styling with CSS variables, Flexbox/Grid, and glassmorphism effects.
- **JavaScript (ES6+)**: Pure JS logic for data management and DOM manipulation.
- **Icons**: [Phosphor Icons](https://phosphoricons.com/) for high-quality, scalable visual elements.
- **Typography**: [Google Fonts (Outfit & Inter)](https://fonts.google.com/) for a sleek, architectural feel.
